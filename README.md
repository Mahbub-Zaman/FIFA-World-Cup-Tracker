# ⚽ FIFA Match Tracker — Chrome Extension

A lightweight and modern Chrome extension that keeps you updated with every FIFA World Cup match directly from your browser.

Whether a match is live, about to start, or already finished, FIFA Tracker provides all the essential information in a clean and intuitive interface.

## ✨ Features

- 🔴 Live Matches
  - Real-time scores
  - Live match status

- 📅 Upcoming Matches
  - Match schedule
  - Kickoff time
  - Team flags
  - Match venue

- ✅ Match Results
  - Final scores
  - Completed match history

- 🕒 Automatic Updates
  - Refreshes match information
  - Displays the last updated time

- 🌙 Modern Dark UI
  - Clean design
  - Lightweight
  - Easy to navigate

## 🚀 How to Use

1. Download or clone this repository.

```bash
git clone https://github.com/your-username/fifa-tracker.git
```

2. Open **Google Chrome**.

3. Navigate to:

```
chrome://extensions
```

4. Enable **Developer Mode** (top-right corner).

5. Click **Load unpacked**.

6. Select the project folder.

7. Pin the extension to your toolbar and enjoy tracking every FIFA World Cup match!
